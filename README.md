# 🎯 POCoS – Punktnotation & Objekt-Coding

Ein interaktiver Zeichen-Editor für den Einstieg in objektorientiertes Programmieren.

👉 Entwickelt für den Einsatz im Unterricht (z.B. 6. Jahrgangsstufe)

---
## 🧠 Idee

Schüler programmieren grafische Objekte mit einfacher Punktnotation:

Ziel:
- Verständnis für Objekte und Attribute
- Einführung in Methoden
- visuelles Lernen durch direkte Rückmeldung

---

## ✨ Features

✅ Interaktiver Code-Editor  
✅ Live-Animation auf Canvas  
✅ Fehlermeldungen mit Hinweisen  
✅ UML-Klassendiagramm mit Tooltips  
✅ Farbpalette zum schnellen Einfügen  
✅ einklappbare Codeblöcke (// Kommentare)  
✅ Gruppierung von Objekten  
✅ Ausführen kompletter oder markierter Programme  

---

## 🧩 Unterstützte Objekte

- Rechteck
- Kreis
- Ellipse
- Dreieck
- Linie
- Gruppe (mehrere Objekte zusammen)

---

## 🎮 Wichtige Methoden

### Bewegung
- `wandern(dx, dy)`
- `WandernBeenden()`

### Physik
- `fallen(y)`
- `huepfen(y)`

### Rotation
- `drehen(winkel)`
- `rotieren(v)`

### Darstellung
- `farbe = "rot"`
- `einblenden()`
- `ausblenden()`

---

## 🛠️ Nutzung

1. Code im Editor eingeben
2. „Ausführen“ klicken
3. Animation beobachten

---

## 💾 Speicherung

- Programme können als `.txt` gespeichert werden
- später wieder ladbar

---

## 🎓 Einsatz im Unterricht

POCOS eignet sich besonders für:

- Einstieg in objektorientiertes Denken
- Visualisierung von Programmabläufen
- kreative Schülerprojekte (Animationen, Spiele)
